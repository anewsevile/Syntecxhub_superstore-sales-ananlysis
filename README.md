## 📌 Project Overview
An end-to-end Sales Analytics Dashboard built using Tableau, 
analyzing the Superstore Sales dataset to uncover business insights 
across products, regions, and time periods.

## 🛠️ Tools Used
- Tableau (Data Visualization)
- Git & GitHub (Version Control)
- Kaggle Superstore Dataset

## 📈 Dashboard Features
- Monthly, Quarterly & Yearly Sales Trends
- Top 10 & Bottom 10 Performing Products
- Region-wise & Category-wise Sales Comparison
- KPI — Total Revenue: $2,261,537
- Interactive Sales Map
