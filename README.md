## 🎯 Objective
Analyze raw sales data to uncover trends, identify top and bottom 
performing products, and present actionable business insights 
through an interactive dashboard.

---

## 🛠️ Tech Stack
| Tool | Purpose |
|------|---------|
| Tableau | Data Visualization & Dashboard |
| Git & GitHub | Version Control |
| Kaggle Superstore Dataset | Raw Sales Data |
